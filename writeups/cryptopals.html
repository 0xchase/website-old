Implement repeating-key XOR

Problem

Write a function that xors a byte string with a multi-byte key. In repeating-key XOR, using the key "ICE", you'll sequentially apply each byte of the key; the first byte of plaintext will be XOR'd against I, the next C, the next E, then I again for the 4th byte, and so on.

Solution

This can be implemented using a single line of python iterators.
